# Leftover Cash Planner

Static GitHub Pages app for routing a default `$2,750` monthly leftover cash pool into editable allocation buckets after bills are paid.

## Features

- Editable monthly leftover cash amount
- Allocation sliders and direct dollar inputs
- Add, rename, remove, and color-code buckets
- Donut and bar charts that update live
- Local browser persistence with reset
- Mobile-first layout for phone use
